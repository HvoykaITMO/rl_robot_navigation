# Проект: Навигация робота на основе RL

## 🎯 Цель
Разработать алгоритм навигации мобильного робота в 2D-пространстве с препятствиями 
с использованием обучения с подкреплением (Reinforcement Learning).

## 📅 Сроки практики
17.06.2026 — 30.06.2026 (14 дней)

## 🧠 Уровень
Новичок в RL. Алгоритм берем готовый (Stable-Baselines3), фокус — на среде и reward shaping.

## 🛠 Стек технологий
- **Язык:** Python 3.10+
- **Среда:** Gymnasium API
- **RL-библиотека:** Stable-Baselines3 (алгоритм PPO)
- **ML-фреймворк:** PyTorch
- **Математика:** NumPy
- **Визуализация:** Matplotlib / Pygame
- **IDE:** Cursor

## 📋 Этапы практики
1. **Аналитический (17-18.06)** — изучение теории, выбор инструментария, проектирование.
2. **Разработка среды (19-22.06)** — Gym-среда, кинематика, сенсоры (ray-casting), визуализация.
3. **Интеграция алгоритма и функция награды (23-25.06)** — подключение SB3, reward shaping.
4. **Обучение и оценка (26-28.06)** — тренировка модели, тесты на новых картах.
5. **Отчетность (29-30.06)** — графики, выводы, оформление отчета.

## 🤝 Правила работы с ИИ
- Я пишу код **самостоятельно**.
- ИИ не дает готовых решений через Ctrl+C / Ctrl+V.
- ИИ выступает в роли ментора: задает наводящие вопросы, указывает на ошибки, проверяет логику.
- Распределение ролей:
  - **Qwen** — архитектор плана и ментор
  - **Perplexity** — поиск документации и быстрые ответы
  - **Claude** — ревью кода (code review)
  - **Cursor** — основная IDE
